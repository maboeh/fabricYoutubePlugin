YouTube to Fabric

Einstellungen und API-Konfiguration

So findest du deinen API Key:

  1. Öffne fabric.so und logge dich ein
  2. Gehe zu Einstellungen in der Fabric App
  3. Generiere einen neuen API Key
  4. Kopiere den Key und füge ihn unten ein

Wichtig: Teile deinen API Key niemals öffentlich!

Hinweis: Der API Key wird lokal im Browser-Speicher abgelegt und ist nur für diese Extension zugänglich.
